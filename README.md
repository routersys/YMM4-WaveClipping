# ウェーブクリッピング for YMM4

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](#)
[![.NET](https://img.shields.io/badge/.NET-10.0-purple.svg)](#)
[![Release](https://img.shields.io/github/v/release/routersys/YMM4-WaveClipping.svg)](https://github.com/routersys/YMM4-WaveClipping/releases)

YukkuriMovieMaker4 (YMM4) 上で動作する、波形マスクによる**映像クリッピング視覚エフェクトプラグイン**です。
サイン波を境界線とした3種類のクリッピングモードにより、映像に動きのある波形トランジションや装飾効果を加えることができます。
HLSLカスタムピクセルシェーダーによる高速なGPU処理と、全パラメータへのキーフレームアニメーション対応を備え、映像制作における表現の幅を大幅に広げます。

## 概要

本プラグインは、YMM4の映像エフェクトとして動作するウェーブクリッピングエフェクトです。
Direct2D1カスタムエフェクトとして実装されたHLSLピクセルシェーダーが、サイン波を境界線としたアルファマスク処理をGPU上でリアルタイムに実行します。
クリップ位置・振幅・周波数・位相を含むすべてのパラメータがYMM4のアニメーションシステムと完全に統合されており、波が流れるトランジションや、リズムに合わせて変化する映像演出をキーフレームで自在に制御できます。

## 主な機能

### 1. 3種類のクリッピングモード

- **一方向**: 単一の波形境界で映像の片側をクリッピングします。波形ワイプトランジションに最適です。
- **双方向**: 2つの波形境界でバンド（帯状領域）を形成し、その内側のみを表示します。波形スリット効果などに活用できます。
- **双方向（反転）**: 双方向モードのクリッピングを反転し、バンドの外側のみを表示します。

### 2. 豊富な調整パラメータ

| パラメータ | 単位 | 初期値 | 範囲 | 説明 |
|---|---|---|---|---|
| クリップ位置 | % | 50 | 0〜100 | 波形境界の垂直位置 |
| バンド幅 | % | 20 | 0〜100 | 双方向モードのバンド幅 |
| 振幅 | % | 5 | 0〜50 | 波の振幅（画像高さ比） |
| 周波数 | - | 3 | 0〜100 | 画像幅あたりの周期数 |
| 位相 | rad | 0 | ±314.159 | 波の水平位相オフセット |
| ソフトネス | - | 0.002 | 0〜1 | 境界エッジの柔らかさ |
| 反転 | - | オフ | オン/オフ | マスクの反転 |

### 3. アニメーション完全統合

クリップ位置・バンド幅・振幅・周波数・位相・ソフトネスのすべてのパラメータがYMM4のアニメーションシステムと統合されており、タイムライン上でキーフレームアニメーションを設定できます。

- **「位相」をアニメーション**: 波が横方向に流れ続けるローリング表現が実現できます。
- **「クリップ位置」をアニメーション**: 波形が画面を横切っていく動的なワイプトランジションを作れます。
- **「振幅」をアニメーション**: 波形が大きくなったり消えたりするダイナミックな演出が可能です。

### 4. HLSLカスタムシェーダーによるGPU処理

- Direct2D1カスタムエフェクトとして実装されたHLSLピクセルシェーダー（`ps_5_0`）が、クリッピング処理をGPU上で高速に実行します。
- `smoothstep` 関数によるアンチエイリアス処理で、ソフトネスパラメータに応じた滑らかなエッジを実現しています。
- プリマルチプライドアルファ形式に対応し、YMM4の合成パイプラインに完全準拠しています。

## 動作要件

- **OS**: Windows 10 / 11 (64bit)
- **YukkuriMovieMaker4**: 最新版を推奨
- **ランタイム**: .NET 10.0

## インストール方法

1. [リリースページ](https://github.com/routersys/YMM4-WaveClipping/releases/latest)から最新の `WaveClipping` プラグインファイル（`.ymme`）をダウンロードしてください。
2. YMM4が起動していないことを確認し、ダウンロードしたファイルを実行（ダブルクリック）してインストールします。
3. YMM4を起動し、映像エフェクトの追加ダイアログを開きます。
4. **「合成」グループ**の中に**「ウェーブクリッピング」**が追加されていますので、それを選択してください。

## 使い方ガイド

### 基本的な使用フロー

1. YMM4のタイムライン上で映像アイテムを選択し、エフェクトの追加を行います。
2. **「合成」グループ**から**「ウェーブクリッピング」**を選択します。
3. エフェクトのプロパティパネルに各パラメータが表示されます。
4. 「モード」で用途に合ったクリッピングモードを選択します。
5. 「振幅」「周波数」「クリップ位置」を調整して、目的の波形形状を設定します。
6. 各パラメータのキーフレームボタンを押してアニメーションを追加します。

### よくある使い方

**波形ワイプトランジション**
「一方向」モードで「クリップ位置」を 0% → 100% にアニメーションさせます。
波形の境界が画面を上から下へと通過するワイプ効果が得られます。

**流れる波形マスク**
「一方向」または「双方向」モードで「位相」を時間に合わせて増加させます。
波形が横方向に流れ続ける動的なマスク表現になります。

**スリット効果**
「双方向」モードで「バンド幅」を小さく設定します。
映像が細い帯状のスリットを通して見える演出ができます。

### 注意事項

- **AviUtl非対応**: 本エフェクトはAviUtl向けの `.exo` エクスポートには対応していません。
- **インターネット通信**: 本プラグインはインターネット通信を行いません。
